# Context-Gathering Checklist

Use this checklist during the **briefing phase** to ensure you capture every dimension needed for a high-quality speech script.

## Speaker Profile
- [ ] Full name and title
- [ ] Organization / company
- [ ] Role in the event (keynote, panel, awards, etc.)
- [ ] Public speaking experience level
- [ ] Known speaking style (data-driven, storytelling, inspirational, etc.)
- [ ] Any verbal tics or preferences to emulate / avoid

## Audience Profile
- [ ] Who is in the room? (title, seniority, function)
- [ ] Audience size (10? 100? 1,000?)
- [ ] What do they already know about the topic?
- [ ] What do they care about most?
- [ ] What is their current sentiment? (skeptical, supportive, neutral)
- [ ] Cultural or language considerations

## Relationship Context
- [ ] Speaker → Audience power dynamic
- [ ] History: first time addressing this group, or recurring?
- [ ] Sensitive topics or landmines to avoid
- [ ] Desired shift in audience perception after the speech

## Event & Format
- [ ] Event name and type (conference, board meeting, town hall, webinar)
- [ ] Slot length (minutes)
- [ ] Format: solo talk, fireside chat, panel, hybrid?
- [ ] Are slides provided? (if yes, how many?)
- [ ] Q&A expected?
- [ ] Will it be recorded / streamed?

## Content Inputs
- [ ] Slide deck or outline (if available)
- [ ] Key messages the speaker MUST land
- [ ] Data points, anecdotes, or quotes to include
- [ ] Any previous speeches or writing samples for style reference
- [ ] Competitor or peer speeches to differentiate from

## Tone & Style
- [ ] Formality level (1-5 scale)
- [ ] Humor: yes / no / light touches only
- [ ] Emotional arc desired (e.g., start serious → end hopeful)
- [ ] Signature phrases or callbacks to use
- [ ] Language: monolingual or bilingual code-switching?

## Logistics
- [ ] Teleprompter, notes on lectern, or memorized?
- [ ] Standing or seated?
- [ ] Microphone type (handheld, lapel, podium)
- [ ] Any props or demos?
