# 餐饮上市公司财报口径字典（高岩版）

适用：连锁餐饮、茶饮咖啡、快餐、火锅等“直营+加盟”混合模式公司。  
目标：避免把“系统交易规模”和“公司确认收入”混为一谈，确保可比性。

---

## 1) 核心概念与标准定义

## 1.1 系统 GMV（System GMV）

- 定义：品牌网络内终端交易总额（通常含加盟店 + 直营店）。
- 典型特征：
  - 反映“品牌交易规模”
  - 不等于公司会计收入
  - 常用于衡量品牌势能、网络规模
- 分析要点：
  - 必须核对是否含税、是否含配送费、是否剔除退款/取消单
  - 必须核对是否仅中国区或含海外

## 1.2 品牌收入 / 公司收入（Revenue / Net Revenues）

- 定义：按会计准则确认到公司报表的收入（GAAP/IFRS）。
- 对加盟模式公司常见构成：
  - 向加盟商销售原料/包材收入
  - 特许权使用费/管理服务费
  - 其他技术/供应链服务费
- 关键关系：
  - **GMV 是交易规模**
  - **Revenue 是公司确认收入**
  - 两者不可直接替代

## 1.3 直营收入 vs 加盟相关收入

- 直营收入（Company-owned）：直营门店直接卖给消费者形成的收入。
- 加盟相关收入（Franchised / Franchise-related）：
  - 加盟店销售并不直接全部计入公司收入
  - 公司通常只确认“供货、服务、管理、特许费”等部分
- 解读原则：
  - 直营占比上升：通常带来更高收入确认、也可能抬高费用与资产负担
  - 加盟占比高：轻资产，但收入与利润对供应链与费率更敏感

## 1.4 同店销售/同店 GMV（SSSG / Same-store GMV growth）

- 定义：同一批“可比门店”在本期与上年同期的销售/GMV增速。
- 可比门店口径常见条件：
  - 开业满 13 个月或 12 个月
  - 比较期内无重大改造/闭店/迁店
- 解读注意：
  - 同店为负不等于总收入为负（可被净开店对冲）
  - 必须看“同店 + 净开店 + 客单 + 交易频次”

## 1.5 开店、闭店、净增、闭店率

- 开店数（Openings）：期间新开门店数量。
- 闭店数（Closures）：期间关停门店数量。
- 净增（Net openings）：开店数 - 闭店数。
- 闭店率（Closure rate）：闭店数 / 期初门店数（或管理口径分母，必须标注）。

推荐统一披露：
- 期初门店、开店、闭店、净增、期末门店（五件套）。

## 1.6 GAAP 与 Non-GAAP

- GAAP 净利润：会计准则口径，含股权激励等项目。
- Non-GAAP 净利润：管理层调整口径，常剔除股权激励、一次性项目。
- 使用规则：
  - 对外结论优先 GAAP
  - Non-GAAP 用于解释经营趋势，但必须给出调节项明细

## 1.7 汇率口径（RMB/美元）

- 境外上市公司常给双币口径。
- 规则：
  - 分析主文统一用人民币（除非用户要求美元）
  - 保留公司披露的换算汇率与日期
  - 跨期对比尽量用同币种，避免汇率噪音

## 2) 常见误判与纠偏

## 误判 A：把 GMV 当成收入

- 错误：GMV 同比 +X%，就说“公司收入 +X%”。
- 正确：分别展示 GMV 增速与 Revenue 增速，并解释差值来源（结构、费率、直营占比）。

## 误判 B：把加盟店终端销售全额并入公司收入

- 错误：加盟网络卖了多少就当公司确认多少。
- 正确：仅按会计政策确认供货/服务/特许费等收入。

## 误判 C：同店下滑=公司经营恶化

- 错误：同店为负就直接下结论“公司衰退”。
- 正确：结合净开店、产品结构、区域扩张与投入阶段判断。

## 误判 D：只看 Non-GAAP，不看 GAAP

- 错误：只引用经调整利润，忽略股权激励与一次性成本。
- 正确：GAAP 与 Non-GAAP并列披露，并解释差异是否可持续。

## 误判 E：闭店率分母口径混乱

- 错误：不同公司直接横向比较闭店率，不核对分母。
- 正确：统一分母（优先期初店数），或明确“不可比”。

## 误判 F：多区域口径混用

- 错误：中国区指标与全球口径混在同一张表。
- 正确：按区域分层（中国/海外/合并），再给 consolidated 结论。

## 3) 高岩标准披露模板（建议）

## 3.1 规模与增长

- GMV（总/分区域）  
- Revenue（总/直营/加盟相关）  
- 同店增速（定义）  
- 门店五件套（期初、开、关、净增、期末）

## 3.2 盈利与效率

- 毛利率、营业利润率、净利率（GAAP）  
- Non-GAAP 利润率（附调节项）  
- 单店产出（如月均 GMV/店、收入/店）

## 3.3 现金与资产质量

- 经营现金流、现金及等价物、受限现金、定期存款  
- 有息负债/租赁负债（如披露）

## 4) 解读写作规则（必须执行）

1. 每条结论至少配 1 个数字证据。  
2. 每个数字后标注口径（GMV/Revenue/GAAP/Non-GAAP/区域）。  
3. 先给“结论句”，再给“证据句”，最后给“策略句”。  
4. 不做“跨口径硬比较”（例如 GMV 利润率）。  
5. 不做“无分母百分比”。涉及占比必须写明分母。  

---

## 5) 餐饮连锁重点字段映射（速查）

- `System GMV` → 系统交易规模（非收入）  
- `Net revenues from franchised stores/teahouses` → 加盟相关确认收入  
- `Net revenues from company-owned stores/teahouses` → 直营确认收入  
- `Same-store sales/GMV growth` → 同店增速  
- `Store openings/closures/net openings` → 开店/闭店/净增  
- `Operating income` → 营业利润  
- `Net income` → 净利润（GAAP）  
- `Non-GAAP net income` → 调整后净利润
